# Configuración de Entorno

## Variables
- `DISCORD_TOKEN`: Token del bot.
- `CLIENT_ID`: ID de la aplicación.
- `GUILD_ID`: ID del gremio para desarrollo.
- `NODE_ENV`: `development` o `production`.
- Supabase opcional: `SUPABASE_URL`, `SUPABASE_ANON_KEY`.

## Node y Discord.js
- Node.js v16.x compatible.
- `discord.js` v14.x.

## Arranque
- `npm run start`.
- En desarrollo, comandos se registran por gremio.
