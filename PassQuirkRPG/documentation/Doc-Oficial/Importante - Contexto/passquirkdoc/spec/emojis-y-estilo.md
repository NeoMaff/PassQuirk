# Emojis y Estilo

## Estilo de Embeds
- Colores temáticos (`config.colors`): primario, éxito, advertencia, error.
- Pies y timestamps consistentes por sistema.

## Emojis
- Usar Unicode por defecto y evitar IDs inválidos.
- Mapear emojis clave: combate, exploración, economía, clases, rarezas.

## Imágenes
- NPCs y banners desde `documentation/Doc-Oficial/Imagenes - Diseño`.
